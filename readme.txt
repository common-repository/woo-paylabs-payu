== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/woo-paylabs-payu' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the PayUmoney WooCommerce payment gateway -> Settings screen to configure the plugin
